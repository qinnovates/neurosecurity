# site/ -- Astro Build Output (GitHub Pages)

This directory is the Astro `publicDir` build output. Do NOT edit files here directly.

## Structure
- `index.html`, `_astro/`, `*.xml`, `rss.xml` -- Generated build artifacts
- `CNAME` -- Custom domain (qinnovate.com)
- `data/` -- Prebuild output (Parquet, JSON, KQL tables). Treat as ephemeral
- `learn/`, `research/`, `atlas/`, `lab/` -- Rendered page directories

## Conventions
- Edit source files in `src/` and run `npm run build` to regenerate
- `site/data/` is regenerated by `npm run prebuild`
- Never commit manual edits to HTML/JS/CSS files in this directory
