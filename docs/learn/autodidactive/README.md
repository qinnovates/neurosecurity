# Autodidactive

A self-directed learning portal for interdisciplinary study across neuroscience, cybersecurity, calculus, quantum physics, neuroethics, and philosophy.

## Launch

**[Open Autodidactive](https://qinnovate.com/learn/autodidactive/)**

## Calculus Labs

Interactive labs connecting calculus fundamentals to BCI security and signal analysis.

| Lab | Description | Link |
|-----|-------------|------|
| Calculus Fundamentals | Limits, derivatives, integrals | [Launch](https://qinnovate.com/learn/autodidactive/labs/calculus-fundamentals.html) |
| Calculus & Signals | Signal processing with calculus | [Launch](https://qinnovate.com/learn/autodidactive/labs/calculus-signals.html) |

## Knowledge Domains

- **Neuroscience** — brain regions, neural signaling, neuroplasticity
- **Cybersecurity** — threat modeling, attack patterns, defense frameworks
- **Calculus** — fundamentals through BCI applications
- **Quantum Physics** — quantum mechanics relevant to neural encryption
- **Neuroethics** — neurorights, consent frameworks, cognitive liberty
- **Philosophy** — key thinkers and ethical frameworks
- **Polymaths** — interdisciplinary thinking patterns

## Auditory Learning

Learn while you work. AI-generated songs and spoken-word content that encode knowledge into formats you can absorb passively. See [auditory/](auditory/).

## Local Development

No build step required. Open `index.html` in a browser or serve locally:

```bash
cd docs/learn/autodidactive
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
