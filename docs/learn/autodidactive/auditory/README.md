# Auditory Learning

Learn while you work. Auditory content designed to absorb passively during deep work, commutes, or downtime.

## Concept

Traditional learning demands your eyes and hands. Auditory learning frees them. The goal: produce AI-generated songs, guided meditations, and spoken-word content that encode real knowledge into formats you can absorb while coding, exercising, or falling asleep.

## How It Works

1. **Research the topic** — distill core concepts into teachable structure
2. **Write lyrics/scripts** — encode the knowledge into memorable, accurate content
3. **Generate with AI** — produce audio using tools like Suno, Mureka, or ElevenLabs
4. **Listen and learn** — passive absorption during other activities

## Why This Works

- **Dual coding theory** — information encoded in multiple modalities (visual + auditory) is retained better
- **Passive exposure** — even background listening builds familiarity with concepts
- **Emotional encoding** — music triggers emotional memory pathways, improving recall
- **Spaced repetition** — songs you replay naturally create spaced review intervals
