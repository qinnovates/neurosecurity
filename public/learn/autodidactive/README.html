What Is This?

Autodidactive is a standalone, mobile-first learning app built with pure HTML, CSS, and JavaScript. No frameworks, no build tools, no server required. Open index.html and start learning.

Each figure includes a full profile: bio, frameworks, habits, struggles, defining moment, quotes, influential books, and actionable takeaways with "Go Deeper" links to Wikipedia, Stanford Encyclopedia, Google Scholar, YouTube, and Internet Archive.

Disciplines

Polymaths (12) Philosophy (7) Neuroethics (6) Neuroscience (6) Quantum Mechanics (8) Cybersecurity (6) Calculus Labs (3)

Features

  • Daily Discovery — seeded PRNG shows one figure per day, same for everyone
  • Spaced Repetition — simplified SM-2 algorithm tracks what you've seen and prompts active recall
  • Note Wall — free-placement post-it notes with drag, color picker, and localStorage persistence
  • Calculus Labs — 3 interactive modules (fundamentals, BCI signals, Fourier analysis) via embedded iframe
  • Bookmarks & Stats — track cards viewed, fields explored, and learning streak
  • PWA Ready — add to home screen on mobile for app-like experience
  • Zero Dependencies — pure ES modules, no npm, no build step, runs from filesystem
