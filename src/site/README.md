# site/ -- Built Output + Static Assets

Files served as-is at the site root. Not processed by Astro's build pipeline.

## Structure

| Directory | Purpose |
|-----------|---------|
| `images/` | Logos, icons, diagrams, screenshots |
| `charts/` | Data visualization assets (PNG/SVG) |
| `models/` | 3D models for Three.js/Babylon.js visualizations |
| `learn/` | Educational content and self-study materials |

## Root Files

| File | Purpose |
|------|---------|
| `robots.txt` | Search engine crawler directives |
| `brain-capture.html` | Standalone brain capture visualization |
